# 🎧 Mikkusu

**Mikkusu** is a lightweight DJ application built in **C++ with JUCE**, designed for smooth track playback, mixing, and experimentation.

> From concept to product — built for real-world testing and continuous improvement.

---

## Features

* Dual deck playback
* Waveform visualization with seek support
* BPM estimation
* Sync between decks
* Hot cues (save, trigger, update)
* 3-band EQ (Low / Mid / High)
* Playlist & track management
* Jog wheel interaction (track nudging)
* Track position and timing display
